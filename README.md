## 说明

本插件可展示的命中相关数据有命中率、弱点命中率、弱点命中次数、命中次数和总弹丸击发数。

散弹枪以弹丸数量计算命中率，穿透弹丸只计算一次命中。

作为主机时可以获取所有玩家的枪械命中率，非主机时可显示所有安装了本插件的玩家的枪械命中率。

未安装本插件的玩家会使用’*‘标注, 且数值可能因网络延迟导致不准确，仅供参考。

## 插件设置

本插件基于 TheArchive，可在 "Settings-->Mod Settings->HUD/UI-->AccuracyTracker" 中对本插件进行个性化设置
